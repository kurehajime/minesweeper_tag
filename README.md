## Installation

```
npm install mine-sweeper-tag
```

## Usage

Define custom element.

```
import { mine_sweeper } from 'mine-sweeper-tag'
customElements.define('mine-sweeper', mine_sweeper)
```

Insert tag.

```
<mine-sweeper cols="10" bomb="10" />
```
